### Project Sharing for Multiple Contributors

Project owners or members with MANAGE permissions can share projects with multiple contributors.

Contributors with VIEW/EDIT permissions can view and modify projects based on their assigned access levels.

### Test Result Storage and Export

Test execution results are stored in the database and can be exported to Excel files containing test case data.

Provides test videos, screenshots, JavaScript source code, and execution logs.

### Database State Updates Post-Testing

This feature enables users to execute SQL queries to update database state after test execution.

### Basic HTTP Authentication Support

This feature allows users to implement Basic HTTP Authentication for user authentication when accessing web applications.

Supports web applications using single Basic HTTP Authentication dialog.

Does not support web applications using multiple Basic HTTP Authentication dialogs.
